# QuickCopy 🚀

A sleek, minimalist, and modern web utility designed to easily extract and copy string values (such as tokens, keys, or text) directly from URL parameters with a single click.

Perfect for quick workflows, API token sharing, and utility redirection.

---


## ✨ Features

* **URL Parameter Parsing:** Automatically reads the `?token=your_text` parameter from the URL.
* **One-Click Copy:** Click either the container box or the button to instantly copy text to your clipboard.
* **Modern UI:** Built with a dark mode aesthetic, smooth hover animations, and glassmorphism touches.
* **Responsive Design:** Optimized for both mobile devices and desktop screens.
* **Zero Dependencies:** Pure HTML, CSS, and vanilla JavaScript. No installation or build steps required.


## 🚀 How to Use

Simply deploy the code (e.g., using GitHub Pages) and pass the text you want to copy inside the URL using the `token` parameter.


### Example URL Structure:
```text
https://your-username.github.io/?token=YourTextHere
